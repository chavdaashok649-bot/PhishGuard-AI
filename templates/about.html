🛡 About PhishGuard AI

PhishGuard AI is a Cyber Security Awareness and Phishing Website Detection platform developed using Python Flask and an AI-based URL analysis approach.

The main purpose of this project is to help users identify suspicious or phishing websites before they share sensitive information such as passwords, banking details, credit card information, or personal data.

Nowadays cyber criminals create fake websites that look almost identical to genuine websites in order to steal user credentials. PhishGuard AI helps reduce this risk by analyzing website URLs and providing a security score with an easy-to-understand result.


🎯 Project Objectives

  • ✔ Detect suspicious and phishing website URLs.
  • ✔ Spread cyber security awareness among users.
  • ✔ Encourage safe internet browsing.
  • ✔ Protect users from phishing attacks.
  • ✔ Help beginners understand cyber threats.
  • ✔ Promote responsible online behaviour.

🔐 How To Keep Your Website Secure

  • 🟢 Always use HTTPS websites.
  • 🟢 Never click unknown email links.
  • 🟢 Verify the website domain before login.
  • 🟢 Use strong and unique passwords.
  • 🟢 Enable Two-Factor Authentication (2FA).
  • 🟢 Keep your browser updated.
  • 🟢 Install trusted antivirus software.
  • 🟢 Never share OTP or banking passwords.
  • 🟢 Avoid public Wi-Fi for financial transactions.
  • 🟢 Regularly update website software and plugins.

⚠ Common Signs of a Phishing Website

  • ❌ Misspelled website domains.
  • ❌ Fake login pages.
  • ❌ Urgent messages asking for passwords.
  • ❌ Suspicious payment requests.
  • ❌ Poor website design and grammar mistakes.
  • ❌ Unknown pop-ups asking for personal information.
  • ❌ Offers that look "too good to be true".

🌍 Why Cyber Security Awareness Matters

Cyber attacks are increasing every year, and phishing remains one of the most common methods used by attackers to steal personal information. By learning to recognize phishing websites and following basic cyber security practices, every internet user can significantly reduce the risk of becoming a victim. Awareness is the first step toward online safety.
